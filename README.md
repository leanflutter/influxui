## Quick Start

### Installation

Add this to your package's pubspec.yaml file:

```yaml
dependencies:
  rise_ui: ^0.1.0
```

### Usage

```dart
import 'package:rise_ui/rise_ui.dart';

void main() async {
  runApp(MyApp());
}
```

> Please see the example app of this plugin for a full example.
